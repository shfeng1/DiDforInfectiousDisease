# Difference-in-Differences for Infectious Disease Outcomes
To use this repo: Start with initializing a Rproject environment “Parallel_Trends_Replication.Rproj”.

Folder “0_Data” contains all data used for replicating our reanalysis of the published studies.

Folder “1a_Scripts” contains all R scripts for replicating simulation experiments.

Folder “1b_Summarize” contains R scripts sumamrizing results from simulations to reprodcue figures and tables (into LaTeX code) in the manuscript.

Folders “2a_School_Masking” and “2b_Kansas_Masking” contain R scripts to reproduce reanalyses of the two published studies. For Kansas masking application, additional county level data was available from COVIDestim at https://covidestim.org.

Folder “3_Temp” to store temporary data files generated from the simulation experiments.

Folder “4_Output” to store all key output files generated from running the codes.
